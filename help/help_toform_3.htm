양식과 데이터 필드 맵핑

양식의 필드와 데이터의 필드를 맵핑합니다.
오른쪽 데이터를 참고하여 왼쪽의 맵핑 정보를 수정하세요

  • 1. 데이터컬럼 : 데이터의 컬럼명 입니다.( 오른쪽 데이터 시트의 머릿글 입니다.-수정불가 )
  • 2. 양식필드명 : 양식파일의 #필드#으로 입력된 필드명 입니다.
    #을 제외한 필드명만 기입하세요 - 양식에 일치하는 필드명이 없으면 무시됩니다.
  • 3. 파일명 : 해당 데이터가 파일명으로 사용될 지 여부 입니다.
    체크되면 해당 데이터가 파일명의 일부로 사용됩니다.
    저장되는 파일명이 중복되지 않는 필드를 선택해야합니다.
    여러 항목의 선택이 가능합니다. - 선택된 항목의 데이터는 파일명의 일부로 사용됩니다.
  • 필드 맵핑이 완료되었다면 "앞으로" 버튼을 클릭하여 다음단계로 이동합니다.
