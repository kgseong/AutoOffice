엑셀시트 합치기

엑셀파일의 두개의 시트를 하나의 시트로 합칩니다.

  • 1. 엑셀파일 선택 : 엑셀파일을 선택합니다.
  • 2. 기준엑셀시트 : 기준이 되는 시트를 선택합니다.
  • 3. 대상엑셀시트 : 기준시트에 합칠 대상 시트를 선택합니다.
  • 기준시트와 대상시트를 선택했다면, 두 시트를 합칠때 일치시킬 컬럼을 선택합니다.
  • 4. 시트간 열 맵핑 정보 : 두 시트를 합칠때 일치시킬 컬럼의 맵핑 정보 입니다.
