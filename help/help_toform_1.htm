양식파일 선택하기

양식으로 사용할 오피스 파일을 선택하세요
양식파일로 엑셀(.xls, .xlsx), 워드(.doc, .docx), 파워포인트(.ppt, .pptx), 한글(.hwpx)파일이 사용 가능합니다

  • 1. 양식파일 선택 : 양식파일을 선택합니다.
  • 2. 양식파일열어보기 : 양식파일 열어 내용을 확인합니다.
  • 3. 엑셀파일을 선택하여 데이터 입력 : 엑셀파일로 부터 데이터를 입력하고자 할때 이 항목을 선택합니다.
    이 항목을 체크하지 않으면 엑셀파일의 내용을 복사하여 데이터를 입력합니다.
  • 양식파일을 선택했다면 "앞으로" 버튼을 클릭하여 다음단계로 이동합니다.

★양식파일의 작성 방법

양식파일 내에 데이터를 입력할 곳에 #필드명#  형태로 입력필드를 작성하세요 
양식파일의  #필드명# 위치에 해당 입력된 데이터의  맵핑된 데이터가 입력됩니다
